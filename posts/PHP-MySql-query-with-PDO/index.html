PHP MySQL query with 「PDO」

Created2022-07-12|Updated2023-03-21|Computer ScienceSQLPHP
|Word count:348|Reading time:2min|Post View:
Don't continue to use original MySQL functions in PHP, let's try PDO !

Why choose PDO ?

First, MySql functions (such mysqli_connect() 、 mysql_connect() ) as its name looks，only for MySQL .
But PDO is more than that !
PDO supported 12 different drivers, which includes MySQL 、 Oracle 、 SQLite 、 MS SQL 、 IBM …and more. (Read documentatin from official for more)

And more importantly, it’s able to effectively prevents SQL Injection .

How to connect ?

It’s needed be the instance.

define('DB',[
  'host' => 'localhost', # the host of database server
  'user' => 'root', # the user of database
  'pass' => 'password', # the password of user
  'name' => 'database', # the name of database
]);

$db_connect = new PDO("mysql:host=".DB['host'].";dbname=".DB['name'].";charset=utf8mb4", DB['user'], DB['pass']);

if(!$db_connect){ die('Database: failed connection !'); } # check if connection is failed

And also recommend that use the follwing settings (just put it below the code above):

# Allow to use the "Prepared Statement".
# It prevents the SQL Injection from happening. (false)
$db_connect->setAttribute(PDO::ATTR_EMULATE_PREPARES, false);

# Whether to prevent convert numeric values to strings when fetching. (false)
$db_connect->setAttribute(PDO::ATTR_STRINGIFY_FETCHES, false);

For more detail, see the PHP official website https://www.php.net/manual/en/pdo.setattribute.php

Pre-Query & Execute

Pre-Query

$sql = "SELECT * FROM `table` WHERE `id`=:id OR `name`=:name;";
$query = $db_connect->prepare($sql, array(PDO::ATTR_CURSOR => PDO::CURSOR_FWDONLY));

Just put the sentence of SQL query to the prepare() function, and the values of sentence are able to use the format :value to represent the value field. (the value in :value can be change the any word you want, except the few sensitive words. And it’s NOT allow to use the duplicate name.)

(It hasn’t not yet be executed from now.)

Execute

$values = [
  ':id' => 123,
  ':name' => 'admin',
];

$result = $query->execute($values);

if($result === false){ die('Query Error !'); };

Put the values to the execute() function (using array format), and it will return the result, also possible to return false when the query got the some errors.

Reference

《PHP: PDO - Manual》https://www.php.net/manual/en/book.pdo.php
