《SQL Basics-03》UPDATE、ALTER、DROP

Created2021-05-25|Updated2021-10-21|Computer ScienceSQLBasics of SQL
|Word count:668|Reading time:2min|Post View:
一些基礎常見的 SQL 語法。

示範的資料

目前資料表內有以下資料：

idphonenameage
10123456789新之助5
2NULLAnonymous19
3NULL向日葵0
40111222333美冴29
50444555666廣志35

如果可以的話也試著一起做做看。

修改

修改資料筆

修改資料筆需要使用「 UPDATE 」語法：

UPDATE `資料表名稱` 
SET `欄位1` = 欄位1新的值, `欄位2` = 欄位2新的值...
WHERE 修改條件;

範例 - 修改資料筆

輸入語法：

UPDATE `資料表名稱` 
SET `name` = "小白", `age` = 0
WHERE `name` LIKE '%onym%';

原資料：

idphonenameage
2NULLAnonymous19

被更改為：

idphonenameage
2NULL小白0

條件「`name` LIKE '%onym%' 」表示模糊查詢 name 欄位「 含有 "onym" 」的資料筆。
ps. 若沒有寫 WHERE 條件，則會更改所有資料。
所有資料的欄位 name 會變「 小白 」，欄位 age 會變「 0 」…

修改欄位

新增欄位

ALTER TABLE `資料表名稱` ADD COLUMN `欄位名稱` 資料型態(長度) 屬性1 屬性2...;

範例 - 新增欄位

ALTER TABLE `資料表名稱` ADD COLUMN `name_jp` varchar(20) NULL;

資料表欄位多出「 name_jp 」：

idphonenameagejp_name
10123456789新之助5NULL
2NULL小白0NULL
3NULL向日葵0NULL
40111222333美冴29NULL
50444555666廣志35NULL

修改欄位

ALTER TABLE `資料表名稱` CHANGE COLUMN `舊欄位名稱`
`新欄位名稱` 新資料形態(長度) 屬性1 屬性2...;

範例 - 修改欄位

ALTER TABLE `資料表名稱` CHANGE COLUMN `phone`
`phone` varchar(11) NULL;

則資料欄位 phone 由型態「 char(10) 」更改至「 varchar(11) 」。

刪除欄位

ALTER TABLE `資料表名稱` DROP COLUMN `欄位名稱`;

範例 - 刪除欄位

ALTER TABLE `資料表名稱` DROP COLUMN `age`;

欄位「 age 」被刪除了：

idphonenamejp_name
10123456789新之助NULL
2NULL小白NULL
3NULL向日葵NULL
40111222333美冴NULL
50444555666廣志NULL

刪除

刪除資料筆

刪除資料筆使用的語法為「 DELETE 」：

DELETE FROM `資料表名稱` WHERE 刪除條件

範例 - 刪除資料筆

DELETE FROM `資料表名稱` WHERE `phone` IS NULL

ps. 「 IS NULL 」這個寫法表示比對值為 NULL (空值) ，等同於「`phone` = "" 」。

原資料為：

idphonenamejp_name
10123456789新之助NULL
2NULL小白NULL
3NULL向日葵NULL
40111222333美冴NULL
50444555666廣志NULL

刪除後：

idphonenamejp_name
10123456789新之助NULL
40111222333美冴NULL
50444555666廣志NULL

當然，如同 UPDATE 語法一樣，沒有指定 WHERE 刪除條件，則會 刪除全部 ！

刪除資料表

刪除資料表使用的是「 DROP 」語法：

DROP TABLE `資料表名稱`;

刪除資料庫

刪除資料庫一樣使用「 DROP 」語法：

DROP DATABASE `資料庫名稱`;
